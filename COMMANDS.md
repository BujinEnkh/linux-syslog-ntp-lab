# Линукс команды

## Syslog
- `syslog` нь системийн өдөр тутмын үйл явдлуудын логыг хадгалахын тулд ашиглагддаг.
- Ж.нь, `logger` командыг ашиглан лог бичиж болно.
- **Жишээ:** `logger "Энэ системийн өдөр тутмын мэдээлэл"`

## NTP (Network Time Protocol)
- `ntpd` үйлчилгээ нь системийн цагийг интернэтэд байгаа цагтай синхрончлохын тулд ашигладаг.
- **Чиглэл:** `ntpdate time.nist.gov`  (NIST цагийн сервер)

## Веб Сервер
- Apache веб серверийг суулгах:  
    `sudo apt install apache2`
- Серверийн статусыг шалгах:  
    `sudo systemctl status apache2`

## Системийн Захиргаа
- Системийн мэдээллийг харах:
    - `top`: процессууд ба системийн ачааллыг харж болно.
    - `df -h`: дискний зайг шалгах.
    - `free -m`: санах ой (RAM) ашиглалт.
